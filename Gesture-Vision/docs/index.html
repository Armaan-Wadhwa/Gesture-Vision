Gesture Guide
  • Pinch right hand = next filter, left hand = previous
  • Open palm (5 fingers) = reset filter
  • Fist (0 fingers) = freeze / unfreeze
  • Thumbs up = save snapshot
  • Thumbs down = discard last snapshot
  • Swipe left/right = switch filter
  • Two hands = filter inside box; thumb distance = intensity
